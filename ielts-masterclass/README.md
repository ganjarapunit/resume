# IELTS Speaking Masterclass

A high-fidelity, interactive e-learning course designed for comprehensive IELTS speaking preparation.

## 🚀 Key Features

- **Standard Compliance**: Full tracking for completion status and success criteria (Pass/Fail) within any standard LMS.
- **AI-Powered Narration**: Model answers narrated by high-quality AI voices for perfect pronunciation and intonation.
- **Interactive Voice Recorder**: Built-in browser recorder allowing students to practice and review their own speaking delivery.
- **WCAG 2.1 AA Compliant**: 
  - High-contrast color palette for visual clarity.
  - Full keyboard navigation support with visible focus states.
  - Semantic HTML5 and ARIA labels for screen reader compatibility.
  - Interactive transcripts for all audio content.
- **Premium Design**: Modern "Glassmorphism" UI built with vanilla CSS, ensuring fast load times and a clean user experience.

## 📦 How to Use

1. **Local Preview**: Open `index.html` in any modern browser.
2. **LMS Deployment**: 
   - Ensure the `assets/` folder contains the generated MP3 files.
   - Use `package_course.py` to create `ielts_speaking_masterclass.zip`.
   - Upload the ZIP to any compatible Learning Management System.
